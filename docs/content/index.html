initialize

Used internally to instantiate an application using provided arguments and returns it.

initialize(application: object, config: object, include: object, modules: object): object
Parameters
application (object) The object on which to call the function.
config (object) Configuration file
include (object) Hashmap of includables ( libraries e.g. ).
modules (object) Hashmap of modules.
Returns
object: An instantiated application

setup

Parameters

config (object) Configuration file

Returns
object: An instantiated application

atomize

Turns objects into atoms.

atomize(target: object): object
Parameters
target (object) Source object
Returns
object: atom Atomized source object
Example
var atom = atomize({foo:'bar'});
atom.get('foo') //returns 'bar';

delegated

Returns a function call to a target object property with one or more prefixed parameters.

delegated(target: object, call: string, argument: any): function
Parameters
target (object) The object on which to call the function.
call (string) Name of the property to call from 'target'.
argument (any) Prefixed argument to use when invoking the method.
Returns
function: A function call which will invoke the method.
Example
//Once the window is loaded, it will call application.onLoaded
//passing 'somedata' as the first argument
window.onload = delegated(application,'onLoaded',somedata)
